# Deen Toolkit by CREATIVE MEN

A modern Islamic web toolkit featuring:
- Hijri date conversion
- Asma-ul-Husna and Asma-un-Nabi ﷺ cards
- Light/Dark theme toggle
- Copy features and elegant UI

## 🛠️ Built With:
- HTML5 + CSS3 (inline and responsive)
- Vanilla JavaScript (no framework)
- Moment.js + Moment Hijri (via CDN)
- Google Fonts: Amiri, Noto Nastaliq Urdu

## 🚀 Hosting Options:
- Fully compatible with Netlify and GitHub Pages
